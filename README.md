# Blog Platform with Admin Panel

This is a full-stack Blog Platform project with an Admin Panel.

## Tech Stack
- Frontend: React (Vite)
- Backend: Python Flask
- Database: SQLite

## Features
### Frontend
- Public blog view
- Admin login
- Admin dashboard
- Create / Edit / Delete blog posts
- User Register / Login
- Comments system UI

### Backend
- Authentication & Authorization (JWT)
- Blog post CRUD APIs
- Comments moderation APIs
- File structure with services/routes (IoC concept)
- API testing using Postman / Hoppscotch
